# echarts4r.maps

215 maps for [echarts4r](https://echarts4r.john-coene.com/).

## Installation

``` r
install.packages("remotes")
remotes::install_github('JohnCoene/echarts4r.maps')
```

## Example

Use in echarts4r

``` r
library(echarts4r)
library(echarts4r.maps)

df <- data.frame(x = "Rajasthan", y = 2)

df %>% 
  e_charts(x) %>%
  em_map("India") %>% 
  e_map(y, map = "India") %>% 
  e_visual_map(y)
```
